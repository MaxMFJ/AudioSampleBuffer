### 3. 灵魂出窍残影效果 👻✨

**核心功能**: `ghostTrail()` 函数

**实现原理**:
```metal
// 5层残影，向左右两侧扩散
for (int t = 0; t < 5; t++) {
    float trailPhase = float(t) / 5.0;
    
    // 残影扩散距离随音频能量变化
    float spreadDistance = trailPhase * 0.08 * audioEnergy;
    
    // 残影透明度逐层衰减
    float trailAlpha = (1.0 - trailPhase) * audioEnergy * 0.6;
    
    // 残影宽度逐渐增大
    float trailWidth = width * (1.0 + trailPhase * 2.0);
}
```

**视觉效果**:
```

弱音频时:
   ░║░       ← 淡淡的残影
```

**特点**:
- 5层残影，由内向外扩散

- 宽度逐层增大 (1x → 1.4x → 1.8x → 2.2x → 2.6x)
- 完全由音频能量驱动，无音频则无残影
